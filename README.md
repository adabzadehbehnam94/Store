این پروژه یک سایت فروشگاهی می باشد 
این پروژه در سایت vercel آپلود شده است .
https://store-seven-taupe.vercel.app/

برای اجرای پروژه به صورت لوکال از کد npm run dev استفاده کنید .
برای ایجاد پوشه node madule از کد npm install استفاده کنید .

در این پروژه از فریم ورک nextjs و لایبرری react استفاده شده . برای سبد خرید از redux استفاده شده .برای استایل دهی از فریمورک bootstrap استفاده شده .قسمت هایی از پروژه مانند قسمت login و register به طور کامل انجام نشده و در کامیت های بعدی تکمیل می شود .
